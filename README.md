# ICMP Pinger Lab
- A Python program that simulates the functionality of an ICMP pinger by sending echo request packets to a specified target host, tracking responses to measure round-trip time, calculating packet loss, and providing detailed output on network connectivity and latency, thereby demonstrating fundamental concepts in network communication and socket programming.
